🖼️ Adding Images

We use the <img> tag to display images in HTML.

🎵 Adding Audio

We use the <audio> tag to play sounds or music. Don’t forget to include the controls attribute so users can play, pause, or stop the sound.

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>

Example:

🎥 Adding Video

To include videos, use the <video> tag and include controls to play, pause, or adjust the volume.

<video controls width="320">
  <source src="movie.mp4" type="video/mp4">
</video>
